# anti-imp

X(Twitter) の **リプライ欄にある「青バッジ(Verified)付き」投稿を非表示**にする Chrome 拡張です。

## 使い方（開発版の読み込み）

1. Chrome を開き、`chrome://extensions` を開く
2. 右上の「デベロッパーモード」を ON
3. 「パッケージ化されていない拡張機能を読み込む」から、この `anti-imp` ディレクトリを選択
4. X を開き、拡張機能アイコン → ポップアップから設定を切り替える

## 機能

- ✅ **リプライ中のブルーバッジアイコンを非表示にする**（デフォルトON）
- ✅ 例外（表示する条件）
  - 「フォロー中の人は表示」
  - 「フォロワー数が閾値以上は表示」

## 注意点 / 仕様

- この拡張は **X の画面構造(DOM)に依存**します。X 側の変更で動かなくなる可能性があります。
- 例外条件（フォロー中/フォロワー数）は X の公式APIを使わずに実現しているため、
  **ユーザーをホバーした時に出るホバーカード等から情報が取れた場合のみ** 反映されます。
  取得できないユーザーは「青バッジなら非表示」になります。

## ファイル構成

- `manifest.json` : Manifest v3
- `background.js` : 初期設定の投入
- `popup/` : ポップアップUI
- `content.js` : X/Twitter ページで DOM を監視して非表示処理
